# Security Policy

## Supported versions

Only the latest release on the `main` branch is supported.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report it privately through GitHub:

- Open a [private vulnerability report](https://github.com/foXaCe/vertical-stack-in-card/security/advisories/new)

Target response time: 7 days.

Since this is a frontend Lovelace card that runs in the browser, please include:

- Home Assistant version
- Browser and version
- A minimal card configuration that reproduces the issue
